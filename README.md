## About the Project

This project is a personal portfolio website built to highlight my web development skills, showcase some of my projects, and provide a way for visitors to learn more about me and my work. This site fulfills the requirements outlined in freeCodeCamp's Responsive Web Design course and is fully responsive.

## User Stories

The following user stories have been implemented:

1. A welcome section with an `id="welcome-section"` that contains an `h1` element.
2. A projects section with an `id="projects"` that holds project details within elements with a class of `project-tile`.
3. Links in the project section that direct to my project pages.
4. A navbar with an `id="navbar"` that contains links to different sections of the page.
5. A link with an `id="profile-link"` that opens my GitHub profile in a new tab.
6. A responsive design with media queries for an optimal viewing experience on various devices.
7. A welcome section height that matches the height of the viewport.
8. A fixed navbar at the top of the viewport.

## Features

- **Responsive Design:** Optimized layout for desktop, tablet, and mobile devices.
- **Easy Navigation:** A fixed navbar that allows for easy navigation between sections.
- **Projects Display:** A dedicated section showcasing a selection of my web development projects.
- **About Me Section:** Information about my skills, interests, and background.
